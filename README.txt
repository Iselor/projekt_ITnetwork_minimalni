Insurance Application
Tento program implementuje aplikaci pro správu pojištěných osob. Uživatel má možnost přidat nového pojištěného, zobrazit seznam všech pojištěných osob a vyhledat konkrétního pojištěného. Aplikace je tvořena na základě zadání minimálních požadavků na závěrečný projekt kurzu "Programátor webových aplikací v Python". 

Instalace
Tento program vyžaduje Python 3.x. Chcete-li spustit tento program, nainstalujte si nejprve Python na svůj počítač.

Spuštění
Stáhněte si tento program do svého počítače.
Otevřete příkazový řádek nebo terminál.
Přejděte do adresáře se staženým programem pomocí příkazu "cd".
Spusťte program pomocí příkazu "python main.py".

Funkce
Přidání pojištěného: Uživatel zadá jméno, příjmení, věk a telefonní číslo nového pojištěného a ten je přidán do seznamu pojištěných osob.
Zobrazení seznamu pojištěných: Program zobrazí seznam všech pojištěných osob.
Vyhledání pojištěného: Uživatel zadá jméno a příjmení pojištěného, kterého hledá, a program vrátí informace o dané osobě, pokud existuje.
Ukončení programu.
